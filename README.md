# UltimaROM-Shamu
Welcome to UltimaROM for the Nexus 6! The first appearance of an UltimaROM on any Nexus device :D

## How-To

To build this ROM yourself, you will need to be using a Linux based operating system.

You can build by following the [Linux building instructions](docs/LINUX.md).
